# 管理员操作指南

更新时间：2026-07-15

本指南面向负责整理解析资料和练习册的机构管理员。当前系统是本机测试模式，开始前请让技术人员保持访问通道开启。

## 登录

1. 在电脑浏览器打开 <http://127.0.0.1:18081/admin>。
2. 输入管理员账号和密码。
3. 登录后会看到三个主要入口。

如果页面打不开，请联系技术人员检查服务和访问通道。不要把管理员密码发送给学生。

## 新建解析二维码

1. 在工作台选择“新建解析二维码”。
2. 填写资料名称、年级和学科。
3. 按需要填写教材版本、章节和备注。
4. 选择答案或讲解文件。
5. 点击“创建解析资料”。

创建后系统会生成资料编号，例如 `QR-Z8SF-ZU3Y`。日常查找可以使用这个短编号，不需要记录页面里的技术编号。

## 下载二维码

进入资料详情页后：

- “下载最新版二维码”会生成自动更新的二维码。以后替换解析文件，旧二维码也会打开新文件。
- “下载锁定当前版本的二维码”会生成固定二维码。它一直打开当前这一个文件版本。

二维码图片可以交给排版人员使用。下载前先阅读页面上的网络状态提示。

## 给练习册添加二维码

1. 在工作台选择“给练习册添加二维码”。
2. 从列表中选择解析资料，可先输入名称或资料编号搜索。
3. 上传要处理的练习册 PDF。
4. 选择“自动更新到最新版”或“锁定当前版本”。
5. 填写需要添加二维码的页码。
6. 选择小、标准或大二维码。
7. 选择左上、右上、左下或右下位置。
8. 当前为本机测试地址时，确认已经了解测试限制。
9. 点击“预览并生成”。
10. 在结果页查看目标页预览，确认位置后下载处理后的 PDF。

## 如何选择最新版或锁定版本

选择“自动更新到最新版”时，二维码适合内容仍会修订的资料。替换文件后，不需要重新印刷二维码。

选择“锁定当前版本”时，二维码适合已经定稿的练习册。以后即使资料更新，这个二维码仍然打开原来的文件。正式印刷前通常应使用锁定版本，并再次核对预览和文件内容。

## 替换解析文件

1. 打开“管理已有解析资料”。
2. 找到资料并点击“替换文件”。
3. 选择新文件，可填写本次变更说明。
4. 确认替换。

替换不会改变资料编号。动态二维码会立即打开新文件；已经锁定的固定二维码不受影响。

## 恢复历史文件

1. 进入资料详情页。
2. 点击“查看历史版本”。
3. 找到要恢复的版本，核对文件名和时间。
4. 点击“恢复为当前版本”并确认。

恢复后，动态二维码会打开被恢复的文件。固定二维码仍打开各自锁定的版本。

## 搜索资料

在“管理已有解析资料”中，可以输入资料名称、资料编号、文件名、教材版本或章节。也可以同时选择年级、学科和状态进行筛选。

找不到资料时，先清空筛选条件，再用资料编号的一部分搜索。已停用资料需要把状态改为“全部状态”或“已停用”才能看到。

## 网络警告

“本机测试地址”表示二维码中的地址是 `127.0.0.1`。电脑通过访问通道可以打开后台，但手机扫码后会访问手机自己，所以无法取得文件。

出现这个警告时，二维码只能用于流程测试，不能正式印刷。只有在技术人员把系统配置为稳定、可达的局域网地址或正式 HTTPS 地址，并完成手机验证后，才能用于相应范围。

## 哪些二维码不能正式印刷

- 页面显示“本机测试地址”的二维码。
- 地址仍可能变化的局域网测试二维码。
- 只允许少量维护人员访问的 Tailscale 二维码。
- 未核对目标页预览、解析内容和版本方式的二维码。
- 已停用或准备删除对应资料的二维码。

## 常见错误

| 提示 | 处理方法 |
| --- | --- |
| 账号或密码不正确 | 检查大小写；连续失败后等待几分钟再试，仍失败请联系技术人员重置密码 |
| 页面已过期 | 刷新页面并重新操作，不要重复提交旧页面 |
| 文件过大或页数过多 | 减小文件，或联系技术人员确认当前限制 |
| PDF 无法读取或已加密 | 使用可正常打开、未加密的 PDF 重新上传 |
| 页码超出范围 | 输入练习册实际存在的页码，第一页填 `1` |
| 二维码区域放不下 | 减小二维码或边距，或换到空白更多的角落 |
| 资料暂时不可用 | 资料已停用，请管理员在详情页重新启用 |
| 手机扫码打不开 | 先看网络提示；本机测试地址下属于正常限制 |

页面出现系统错误且刷新后仍存在时，记录资料编号、操作时间和页面提示，交给技术人员排查。不要把管理员密码或完整会话信息放进截图。

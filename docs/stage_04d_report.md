Stage 4D 已增加 PNG、JPEG、WebP 图片答案和默认关闭的受控外部网页答案。PDF、草稿发布、并发保护、固定二维码和旧入口保持兼容。未修改网络、Tailscale、防火墙或 QuickDrop。

## 图片内容

- 支持 PNG、JPEG、WebP，真实 MIME 由 Pillow 检测。
- 校验真实字节大小、完整解码、格式、宽高和最大像素数。
- 损坏图片、伪装格式、SVG、GIF、超限字节和超限像素均拒绝。
- 图片先成为草稿，可在管理员页面直接预览。
- 发布后学生页立即显示自适应图片，可查看原图和下载。
- 不可变缓存、SHA-256 ETag、中文文件名和固定二维码均通过测试。

## 外部地址

- 默认 `ALLOW_EXTERNAL_URLS=false`，后台不显示入口。
- 显式开启后默认只允许 HTTPS，可配置域名白名单和黑名单。
- localhost、私网、链路本地、组播、Tailscale 和危险协议默认拒绝。
- 服务端只做标准 URL、DNS 和 IP 分类校验，不发出 HTTP 请求。
- 外部地址同样创建草稿，发布前要求管理员确认。
- 学生页不嵌入、不自动跳转；点击后返回 307。
- 请求参数不能覆盖跳转目标。
- 固定外链二维码和历史重新发布通过测试。
- 外链校验失败审计只记录域名，不记录敏感 query。

## 遗留风险

- DNS rebinding 无法仅靠应用层校验彻底消除，正式环境应使用机构控制的 HTTPS 白名单。
- 外部网站内容可随时变化、失效或增加追踪，系统不保存其快照。
- 当前没有全局外链清单、定期失效检查或外链撤销工作流。
- 图片不生成缩略图，大图虽然有像素和字节限制，学生首次加载仍取原图。

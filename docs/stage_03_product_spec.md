# 第三阶段产品说明

更新时间：2026-07-15

## 目标用户

系统面向学校、培训机构或出版社中负责整理练习册和解析资料的普通管理员。管理员不需要知道二维码内部编号、接口地址或数据库结构，也不需要使用 Swagger。

学生只使用印刷在练习册上的二维码。学生入口是只读的，不提供上传、替换、回滚或管理能力。

## 中文产品术语

| 页面用语 | 含义 |
| --- | --- |
| 解析资料 | 答案、讲解或其他需要由二维码打开的文件 |
| 资料编号 | 系统生成的 `QR-XXXX-XXXX` 短编号，用于搜索和人工识别 |
| 自动更新到最新版 | 动态二维码，始终打开资料的当前版本 |
| 锁定当前版本 | 固定二维码，永久打开生成时选定的版本 |
| 替换解析文件 | 上传新文件并把它设为当前版本 |
| 恢复历史文件 | 把一个历史版本重新设为当前版本 |
| 停用资料 | 暂停学生入口，但不删除文件和历史记录 |

`qr_id`、`version_id`、SHA-256 和实际 URL 只放在折叠的“技术信息”中，供排障使用。

## 页面结构

- `/admin/login`：中文管理员登录页。
- `/admin`：工作台，只突出“新建解析二维码”“给练习册添加二维码”“管理已有解析资料”三个入口。
- `/admin/materials`：资料搜索、筛选和分页列表。
- `/admin/materials/new`：上传解析资料并填写名称、年级、学科、教材版本和章节。
- `/admin/materials/{id}`：资料详情、动态二维码、固定二维码和常用操作。
- `/admin/materials/{id}/replace`：替换当前解析文件。
- `/admin/materials/{id}/versions`：查看并恢复历史版本。
- `/admin/materials/{id}/edit`：编辑业务信息。
- `/admin/pdf/new`：给练习册添加二维码的分步向导。
- `/admin/pdf/jobs/{id}`：目标页预览、生成结果和 PDF 下载。

## 三个主要流程

### 新建解析二维码

管理员填写资料名称、年级和学科，上传答案或讲解文件。系统生成唯一资料编号和动态二维码，并进入资料详情页。

### 给练习册添加二维码

管理员从列表选择已有解析资料，上传练习册 PDF，选择动态或固定方式、页码、四角位置和大小。系统生成带二维码的 PDF，并显示目标页预览和下载按钮。

### 管理已有解析资料

管理员可按名称、资料编号、文件名、教材或章节搜索，并按年级、学科、状态筛选。详情页支持修改资料信息、替换文件、恢复历史版本和停用或启用资料。

## 两种二维码

动态二维码使用 `/r/{qr_id}`。替换或恢复文件后，已经印刷的二维码会跟随当前版本变化，适合持续维护的解析资料。

固定二维码使用 `/r/{qr_id}/versions/{version_id}`。它始终指向生成时选定的文件版本，适合已经定稿并准备正式印刷的内容。被固定二维码引用的版本会被标记并排除在自动清理之外。

## 管理员可见信息

- 资料名称、资料编号、年级、学科、教材版本和章节。
- 当前文件名、大小、版本号、更新时间和启用状态。
- 动态或固定方式的中文解释。
- 历史版本、是否当前版本、是否受固定二维码保护。
- PDF 目标页、二维码位置、大小、生成状态和预览。
- 当前网络是否适合手机扫码及是否允许正式印刷。

## 默认隐藏的技术信息

- `qr_id`、`version_id` 和 PDF job ID。
- 动态 URL、固定版本 URL。
- MIME、字节数和 SHA-256。

这些字段集中在折叠的“技术信息”区域，不参与普通操作。

## 安全边界

- 管理页面必须登录；状态修改表单使用 CSRF 防护。
- 管理 API 在生产配置中需要管理员会话或可选 Bearer 令牌。
- 管理员会话使用签名、`HttpOnly`、`SameSite=Lax` Cookie，并有有效期。
- Swagger 和 OpenAPI 默认关闭。
- 学生入口只允许读取动态或固定版本文件；停用资料返回中文不可用页。
- 服务默认仅绑定 `127.0.0.1`，不会自动开放局域网、Tailscale 或公网。
- 上传文件使用内部生成的存储名，原文件名仅作为经过清洗的显示和下载元数据。

## 当前不实现

- 多管理员、角色、权限矩阵和审计日志。
- 批量建资料、批量盖码、ZIP 导出和一个 PDF 放多个二维码。
- 自由拖拽坐标、PDF 合并和在线编辑 PDF 内容。
- 学生账号、扫码统计、个人信息收集和学习分析。
- 公网域名、HTTPS、反向代理、自动备份恢复和高可用部署。
- 自动修改防火墙、Tailscale ACL 或服务监听地址。
